API Reference

Key Generation

Endpoint: POST /key_gen

Description: Generate a new ECDSA key pair

Request

{
    "network": 4, // Example network ID
    "request_id": "optional-request-id" // Optional
}
Field Type Description
network number The network ID for which the key is being generated
request_id string Optional unique identifier for the request

Response

{
    "data": {
        "request_id": "...",
        "address": "0x...",
        "duration": 0.123
    }
}
Field Type Description
data.request_id string Unique identifier for the request
data.address string The generated Ethereum address
data.duration number Time taken to generate the key pair (in seconds)

Sign

Endpoint: POST /sign

Description: Sign a message using the generated key

Request

{
    "address": "0x...",
    "tx_origin": "..."
}
Field Type Description
address string The address associated with the key
tx_origin string (encoded base64) The transaction origin data to be signed

Response

{
    "data": {
        "v": "...",
        "r": "...",
        "s": "..."
    }
}
Field Type Description
data.v string The v component of the ECDSA signature
data.r string The r component of the ECDSA signature
data.s string The s component of the ECDSA signature

Get All Networks

Endpoint: GET /networks

Description: Retrieve all available networks

Request

This endpoint doesn't require any parameters.

Response

{
    "data": [
        {
            "name": "Ethereum",
            "chain_id": 1
        },
        ...
    ]
}
Field Type Description
data[].name string The name of the network
data[].chain_id number The chain ID of the network

Error Codes

TECDSA API uses the following error codes:

Error Code HTTP Status Description
BAD_REQUEST 400 잘못된 요청입니다
UNAUTHORIZED 401 인증되지 않은 요청입니다
FORBIDDEN 403 접근이 금지되었습니다
NOT_FOUND 404 요청한 리소스를 찾을 수 없습니다
INTERNAL_SERVER_ERROR 500 내부 서버 오류가 발생했습니다
KEY_GENERATION_ERROR 500 키 생성 중 알 수 없는 오류가 발생했습니다
SIGNING_ERROR 500 서명 중 오류가 발생했습니다

Additional Error Messages

  • 잘못된 요청 본문입니다
  • 지원되지 않는 네트워크입니다
  • 중복된 요청 ID입니다
  • 클라이언트 보안 정보를 가져오는데 실패했습니다
  • 스트림 설정에 실패했습니다
  • 키 생성 시작에 실패했습니다
  • 키 생성 중 실패했습니다
  • 유효하지 않은 요청 ID입니다
  • gRPC 연결에 실패했습니다
  • 서명 요청이 유효하지 않습니다
  • 서명 프로세스 시작에 실패했습니다
  • 서명 프로세스 중 실패했습니다
